# 微信机器人

## 免开发运行

1. 下载release中的压缩包，解压后点击 `启动.cmd` 运行
2. 运行前启动登录微信

[详细教程](https://pug.minapp.xin/doc/help)

## 功能介绍

1. 群统计 - 发言统计，邀请统计
2. 群积分(开发中)
3. 群收藏
4. 群接口(开发中)
